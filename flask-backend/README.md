# Using this flask backend for testing

## Setup

- Install Python 3.6+
- (Optional) Install virtualenv / venv or equivalent (recommended)
- From inside your virtual environment: `pip install -r requirements.txt`
- If not using a virtualenv: `pip3 install -r requirements.txt`

## Running

`python backend.py`

Starts up a server on port 5000. In debug mode, any changes made to the code will be detected and the server restarts itself.
